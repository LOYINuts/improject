# IM即时通讯系统

## 技术栈

**语言:**`Golang`

**数据库:**`MySql`

**框架:**`GIN`,`GORM`

**协议:**`Websocket`

## 基本介绍

一个基于`GO`,`GIN`,`GORM`,`Websocket`实现的即时通讯系统。用户在房间里聊天，私聊属于两个人的房间，体现出好友关系，群聊不体现。使用`Websocket`协议发送接收消息

## 功能列表

- 用户登录
- 用户注册
- 用户更新(昵称(NickName),性别(gender))
- 邮件发送(用于绑定，解绑邮箱以及改密)

- 核心功能:发送接受消息
- 获取聊天记录
